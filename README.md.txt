# Basketball website

## This webapp displays all the current data about players,leagues and fixtures.

## Tech Stack used --
* HTML
* CSS
* JavaScript
* jQuery
* BootStrap
* Ajax

## API Endpoints used --
* [API to access league details](https://allsportsapi.com/basketball-api-documentation#Leagues)
* [API to access livescores](https://allsportsapi.com/basketball-api-documentation#Livescore)
* [Api to access current Standings](https://allsportsapi.com/basketball-api-documentation#Standings)
* [Api to access Team](https://allsportsapi.com/basketball-api-documentation#Teams)
